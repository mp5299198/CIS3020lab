Reflection on Building This Website

Creating this multi‑page Legend of Zelda fandom website has been a great opportunity to practice layout design, navigation structure, and consistent styling across multiple pages. One of the biggest improvements in this version of the site was implementing a shared external CSS file, which allowed for a unified look and feel throughout the entire project.

I chose the Royal Hyrule theme because the deep navy and gold color palette reflects the regal atmosphere of Hyrule Castle and the long‑standing legacy of the Zelda series. The design focuses on readability, clean spacing, and subtle decorative elements that enhance the content without overwhelming it.

What I Learned

  • How to use a shared external CSS file across multiple pages
  • How to create a responsive layout using flexible containers and media queries
  • How to design consistent navigation and footers
  • How to apply color theory to create a cohesive theme
  • How to structure content for readability and accessibility

Challenges and Solutions

One challenge was ensuring that the layout remained readable on smaller screens. To solve this, I used a max‑width container, percentage‑based spacing, and a media query that adjusts the navigation layout on mobile devices.

Another challenge was maintaining visual consistency across all pages. Using shared classes like .container and .box helped keep the structure uniform and easy to manage.
